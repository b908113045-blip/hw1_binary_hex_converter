# 作業一：進位轉換器 (Binary/Decimal/Hex Converter)

這是一個基於 Python Tkinter 開發的圖形化介面程式，支援二進位、十進位與十六進位之間的相互轉換。

## 專案資訊
* **專案名稱**：hw1_binary_hex_converter
* **開發語言**：Python 3.x
* **GUI 框架**：Tkinter

## 功能亮點
1.  **完全手動轉換邏輯**：本程式「不使用」任何程式語言內建的進位轉換函式（如 Python 的 `bin()`, `hex()`, `int(x, 2)` 等）。所有的轉換皆透過數學運算與 ASCII 碼偏移量手動實現。
2.  **超越範圍支援（加分項目）**：程式不限制於 0-255 的範圍，支援任意大小的正整數轉換。
3.  **錯誤處理**：若輸入不符合進位規範的字元（例如在二進位框輸入 `2`），程式會跳出錯誤提示視窗。

## 轉換邏輯說明

為了符合課程要求，本程式核心轉換邏輯如下：

### 1. 任意進位轉十進位 (Manual Base to Decimal)
我們透過疊加法來計算。對於輸入字串中的每個字元：
* 若是 `0-9`，數值為 `ord(char) - ord('0')`。
* 若是 `A-F`，數值為 `ord(char) - ord('A') + 10`。
* 將當前總計值乘上原進位基數，再加上新讀取的數值，直到字串結尾。

### 2. 十進位轉任意進位 (Manual Decimal to Base)
我們使用「除法取餘數」法：
* 將十進位數值不斷除以目標進位基數（2 或 16）。
* 每次取得的餘數對應到字元表 `0123456789ABCDEF`。
* 將餘數對應的字元逆序排列，即得到目標進位字串。

## 安裝與執行

### 環境需求
* Python 3.10+
* Tkinter 函式庫

### 執行程式
在終端機進入專案目錄後執行：
```bash
python3 main.py
```

## 使用說明
1.  在 **Binary**、**Decimal** 或 **Hexadecimal** 任意一個輸入框輸入數值。
2.  點擊 **Convert** 按鈕，其他兩個輸入框會自動顯示對應的轉換結果。
3.  點擊 **Clear** 按鈕可清空所有數值，重新開始。
